# 场景：持续学习

> 版本：v1.0 | 更新：2026-04-23

## 场景定义
作为 AI/科技领域博主，需要持续学习新知识、新工具、新趋势，并将学习成果转化为内容。

## 目标
- 每周至少深度学习 1 个新主题
- 学习成果 80% 转化为内容输出
- 建立系统化的知识体系

## 资源映射

| 层级 | 使用的资源 |
|------|-----------|
| 身份层 | current_projects.md（了解当前学习方向） |
| 知识层 | decisions/（学习笔记和决策记录）、materials/（收集的素材） |
| 工具层 | research-topic Skill |
| 协作层 | 自定义学习工作流 |

## 学习闭环

```
[发现新主题]
    ↓
AI 搜索 + 整理信息 → research-topic Skill
    ↓
阅读/体验 + 做笔记 → 沉淀到 decisions/
    ↓
思考: 这能变成什么内容？
    ↓
[进入内容生产管线] → 复用 content-production 场景
    ↓
发布后复盘 → 更新知识库
```

## 度量指标
- 每周学习主题数
- 学习→内容转化率
- 知识库增长量
