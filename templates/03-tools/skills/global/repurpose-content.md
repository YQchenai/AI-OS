# 内容多平台改编

> 版本：v1.0 | 更新：2026-04-23 | 使用次数：0

## 触发条件
有一篇内容需要改编为其他平台版本时使用。

## 输入
- 原始内容（或指向原始内容的路径）
- 目标平台：小红书 / 公众号

## 输出
- 改编后的内容
- 平台适配说明

## 执行步骤

### 公众号 → 小红书
1. 提取核心观点和 3-5 个关键信息点
2. 压缩至 300-800 字
3. 改写为口语化表达
4. 添加 emoji 适度点缀
5. 生成小红书标题和标签
6. 建议配图方案

### 小红书 → 公众号
1. 扩展每个要点，补充论据和案例
2. 增加开头故事/引入
3. 增加结尾升华
4. 调整为公众号排版风格
5. 生成公众号标题和摘要

## 注意事项
- 改编不是简单缩放，要适配平台调性
- 小红书重"快+实用"，公众号重"深+观点"
- 保留核心观点，调整表达方式
