# 去 AI 腔调

> 版本：v2.0 | 更新：2026-04-24

## 触发条件

AI 生成的文案有明显的"AI味"时使用。建议在所有写作 Skill 的最后一步自动调用。

## 输入

- AI 生成的文案
- `01-identity/memory/writing_style.md` 中的风格标本

## 输出

- 去 AI 腔后的文案

## 执行步骤

---

### L1 硬性规则（必须修复）

逐条扫描，发现一条修一条，不允许跳过。

#### 1.1 禁用词替换表

| 禁用 | 替换为 |
|------|--------|
| 说白了 | 坦率的讲、其实就是 |
| 意味着什么 / 这意味着 | 那结果会怎样呢、所以呢 |
| 本质上 | 说到底、其实 |
| 换句话说 | 你想想看、也就是说 |
| 不可否认 | 直接删掉 |
| 综上所述 / 总的来说 | 换成具体的回扣句 |
| 首先...其次...最后 | 用自然转场词替代 |
| 值得注意的是 / 不难发现 | 删掉，直接说 |
| 让我们一起 | 咱们来 |
| 在当今社会 | 删掉，直接进入主题 |
| 随着XX的发展 | 删掉，直接说结论 |
| 旨在 | 为了 |
| 赋能 | 帮助/支持 |
| 颗粒度 | 细节 |
| 闭环 | 完整流程 |
| 打法 | 方法/策略 |
| 沉淀 | 积累/总结 |

#### 1.2 禁用标点

| 禁用 | 处理方式 |
|------|----------|
| 冒号「：」 | 用逗号代替 |
| 破折号「——」 | 用逗号或句号 |
| 双引号「""」「"」」 | 用「」或直接不加 |

#### 1.3 结构性套话扫描

以下模式必须识别并修复：

- 「让我们来看看」「接下来让我们」→ 删掉或换成口语化转场
- 「在当今...的时代」「随着...的发展」→ 删掉，直接进入主题
- 连续 bullet point 罗列超过 3 个 → 改成自然段落叙述
- 大段加粗超过 2 行 → 拆开，只加粗关键词

#### 1.4 空泛工具名检查

- 「AI工具」「某个模型」「一款软件」→ 必须写出具体名字
- 「某个平台」「某个网站」→ 写出具体名称

---

### L2 风格修复（注入活人感）

L1 修完之后，文章可能"干净了"但"没灵魂"。L2 的任务是让文字像人写的。

#### 2.1 读取风格标本

- 读取 `01-identity/memory/writing_style.md` 中的风格标本
- 提取标本中的句式特征、常用表达、节奏模式
- 用标本的语气校准当前文案

#### 2.2 注入口语化表达

从推荐词组库中选取合适的表达，自然嵌入文章。不要堆砌，全文 8-10 处即可。

**完整推荐口语化词组库：**

**转场过渡：**
坦率的讲、说真的、我是真的觉得、反正我觉得、怎么说呢、其实吧、你想想看、回到xxx这块、说到这里、话又说回来

**表达判断：**
我有时候觉得、我一直觉得、不是说xxx不行而是说、我自己的感受是、以我的经验来看、至少对我来说

**承认自嘲：**
说实话我也不确定、我自己也还在摸索、可能有些想法还不成熟、愚钝如我、我也不敢说一定对、这个我自己也踩过坑

**情绪表达：**
这种感觉太爽了、我当时就愣住了、太离谱了、给我一下子整不会了、说实话有点上头、看完真的沉默了

**拉近距离：**
很多朋友可能不知道、可能有小伙伴纳闷、大家也都知道、你们应该也有这种感觉、不知道你们有没有遇到过

#### 2.3 制造节奏变化

- 长短句交替，不要全是长句也不要全是短句
- 一句话独立成段制造重点，全文至少 3 次
- 用逗号制造口语化停顿，模拟说话的节奏
- 偏离主线后用"扣主线句"拉回来

#### 2.4 加入自嘲或承认不足

至少一处，让作者显得真实而非全知全能。例如：

- 「说实话我也不确定这个判断对不对」
- 「我自己也还在摸索这个方向」
- 「可能有些想法还不成熟，先抛出来大家聊聊」

---

### L3 最终检查

通读全文，问自己三个问题：

1. **这像真人写的吗？** 如果不像，回到 L2 继续调整
2. **读起来顺吗？** 出声读一遍，哪里磕巴改哪里
3. **有没有遗漏的 AI 腔？** 回到 L1 再扫一遍

## 注意事项

- 去 AI 腔不是让文字变"糙"，而是让文字有"人味"
- 保留专业性，去掉机器感
- 每个人的"人味"不同，要参考风格标本来校准
- L1 是硬性底线，L2 是风格提升，L3 是最终兜底，三层缺一不可
- 禁用词表不是终点，遇到其他"听起来像 AI 写的"表达也要主动替换
